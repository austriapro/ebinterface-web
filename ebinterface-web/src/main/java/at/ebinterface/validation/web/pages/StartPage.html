Beschreibung


Das ebInterface Prüfservice dient zum Validieren von ebInterface 3.0, 3.02, 4.0 und 4.1 Instanzen. Dabei wird die Gültigkeit der Instanz geben das entsprechende Schema geprüft, sowie bei Vorhandensein einer digitalen Signatur auch deren Gültigkeit. Für ebInterface 4.0/4.1 ist auch das Prüfen von zusätzlichen Regeln, welchen eine ebInterface 4.0/4.1 XML Rechnungsinstanz genügen muss, möglich. Zusätzlich haben Sie die Möglichkeit ebInterface XML Instanzen in einem ausdruckbaren Format anzeigen zu lassen.
