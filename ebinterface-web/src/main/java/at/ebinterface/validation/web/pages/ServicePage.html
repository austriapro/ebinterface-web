Auf dieser Seite finden Sie die wichtigsten Services rund um die E-Rechnung im Format ebInterface.

ebinterface Rechnung erstellen

Wenn Sie keine Möglichkeit haben, mit Ihrer eigenen Software ebInterface Rechnungen zu erstellen bzw. nur selten Rechnungen im Format ebInterface benötigen, können Sie unser Online-Rechnungsformular verwenden. Software-Hersteller und Dienstleister, die nach eigenen Angaben den Rechnungsstandard ebInterface in ihren Produkten bzw. Services umgesetzt haben, finden Sie unter partner.ebinterface.at.

Rechnung online erstellen

Dieser Button führt Sie direkt zum ebInterface Online-Formular.

ebInterface Rechnung prüfen und als PDF anzeigen

Dieses Tool prüft die Gültigkeit der ebInterface Rechnung und wandelt sie in eine leicht lesbare PDF-Datei um. Der QR-Code, der bei dieser Konvertierung auf der Rechnung generiert wird, ermöglicht die automatische Übernahme der Daten ins Mobile Banking des Rechnungsempfängers.
